# ProyectoBanco - Alke Wallet
Proyecto web educativo que simula el funcionamiento básico de una billetera digital / banca en línea. Permite iniciar sesión, visualizar información de cuenta, realizar depósitos, transferencias y revisar el historial de movimientos, todo utilizando HTML, CSS, JavaScript y Bootstrap, con persistencia de datos mediante localStorage.
